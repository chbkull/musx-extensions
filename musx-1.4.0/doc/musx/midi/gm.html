Module musx.midi.gm

Defines constants for GM instruments, controllers and drum map values, and dictionaries for reverse mapping to names. Names are sanitized for Python syntax but are close to the names listed in at https://www.midi.org/specifications-old/item/general-midi-2

Global variables

var controller_names

A dictionary of GM controller names indexed by controller change values 0 to 127. Note: these are the true midi values, i.e. one less than the one-based numbers listed in GM tables on the web. There are a number of undefined values in the range so you should iterate using controller_names.items()

var instrument_names

A dictionary of GM instrument names indexed by program change values 0 to 127. Note: these are the true midi values, i.e. one less than the one-based numbers listed in GM tables on the web.

var percussion_names

A dictionary of GM drum map names indexed by controller change values 34 to 80. Note: these are the true midi values, i.e. one less than the one-based numbers listed in GM tables on the web.
